Plataforma web enfocada en transformar procesos manuales de evaluación en un flujo digital, claro y profesional.

Convierte la captura de datos de candidatos en reportes PDF listos para entregar, con estructura corporativa, identidad visual y anexos documentales.

## ¿Qué hace esta página?

- Centraliza la captura de información socioeconómica en una sola experiencia.
- Ordena y valida grandes volúmenes de datos personales, familiares, académicos y laborales.
- Genera automáticamente un PDF formal con diseño homogéneo y encabezados institucionales.
- Integra fotos del candidato y del domicilio dentro del reporte final.
- Fusiona documentos PDF adicionales para consolidar todo en un solo archivo.

## Lo más destacado

- Formulario inteligente con secciones dinámicas para registrar múltiples entradas sin fricción.
- Documento final consistente, limpio y listo para revisión o archivo.
- Marca de agua y branding para reforzar presentación profesional.
- Flujo pensado para equipos de reclutamiento, recursos humanos y validación documental.

## Valor que entrega

- Reduce tiempos operativos al eliminar armado manual de reportes.
- Disminuye errores de captura mediante una estructura guiada.
- Mejora la trazabilidad de cada evaluación en un formato uniforme.
- Eleva la calidad visual y técnica de la entrega al cliente interno o externo.

## Stack tecnológico

- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Generación de reportes: EJS + Puppeteer + pdf-lib
- Despliegue: Docker

## Resultado

Una solución completa para digitalizar el estudio socioeconómico de principio a fin, desde la captura hasta la emisión de un reporte profesional, consolidado y listo para compartir.
